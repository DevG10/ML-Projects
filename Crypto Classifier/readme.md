The Crypto Classifier is a machine learning pipeline designed to accurately classify cryptocurrency addresses into various types. By leveraging machine learning algorithms and data preprocessing techniques, this pipeline offers robust classification capabilities for a wide range of cryptocurrency types.

## Introduction

The Crypto Classifier is a machine learning pipeline designed to accurately classify cryptocurrency addresses into various types. By leveraging machine learning algorithms and data preprocessing techniques, this pipeline offers robust classification capabilities for a wide range of cryptocurrency types.

## Features

- Data collection from Google's open source dataset
- Data preprocessing including cleaning and feature engineering
- Model selection with support for multiple algorithms
- Training and evaluation of machine learning models
- Deployment of trained models for inference

## Requirements

- Python 3.x
- pandas
- scikit-learn
- xgboost
- Streamlit

Install required libraries using pip:

```bash
pip install pandas scikit-learn xgboost streamlit
```
## Installation

To install the Crypto Classifier, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/DevG10/ML-Projects.git
    cd ML-Projects
    ```
2. Install dependencies
   ```bash
   pip install -r requirements.txt
    ```
## Usage
1. Navigate to the project directory:
    ```bash
    cd ML-Projects/Crypto Classifier
    ```
2. Run the streamlit app
  
     ```bash
    streamlit run crypto_app.py
    ```
3. Access the app in your browser at http://localhost:8501.
